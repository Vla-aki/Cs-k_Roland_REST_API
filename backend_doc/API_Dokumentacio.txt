# TÚRA PORTÁL BACKEND - DOKUMENTÁCIÓ

## TECHNOLÓGIÁK:
- Node.js + Express (szerver)
- MySQL (adatbázis)
- Két változat: 1) Natív MySQL 2) Sequelize ORM
- CORS, dotenv (segédkönyvtárak)

## 7 MODELL (TÁBLA):
1. felhasznalok - felhasználók adatai
2. turavezetok - túravezetők  
3. programok - fizetős túrák
4. nyitott_programok - ingyenes túrák
5. foglalasok - ki mit foglalt (pivot N:M-hez)
6. ertekelesek - értékelések (pivot N:M-hez)
7. program_kategoria - kategóriák (pivot N:M-hez)

## FŐ VÉGPONTOK:

1. FELHASZNÁLÓK:
GET    /api/felhasznalok              # Összes felhasználó
POST   /api/felhasznalok              # Új felhasználó
GET    /api/felhasznalok/:id          # Egy felhasználó

2. PROGRAMOK:
GET    /api/programok                 # Fizetős túrák
GET    /api/nyitott-programok         # Ingyenes túrák

3. FOGLALÁSOK:
GET    /api/foglalasok                # Összes foglalás
POST   /api/foglalasok                # Új foglalás

4. TÖBB-TÁBLÁS LEKÉRDEZÉSEK (fontos!):
GET /api/felhasznalok/:id/foglalasok  # Felhasználó összes foglalása
GET /api/programok-reszletes          # Programok kategóriákkal

## INDÍTÁS:
MySQL:      npm run dev       (http://localhost:3000)
Sequelize:  node server_sequelize.js  (http://localhost:3001)
